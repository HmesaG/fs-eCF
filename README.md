# fs-eCF
Plugin de Facturación Electrónica e-CF para República Dominicana (DGII) en FacturaScripts.

## Características
- Firma digital XMLDSig (RSA-SHA256).
- Comunicación con la API REST de DGII.
- Gestión de TrackId y polling de estado.
- Soporte para comprobantes de crédito fiscal y consumidor final.

## Instalación
Subir el archivo `eCF_GMV.zip` desde el panel de control de FacturaScripts.
